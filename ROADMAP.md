# Roadmap

This roadmap tracks public work around snapmog.com and the SnapMog product surface.

## Now

- Improve first-run clarity for users who understand Omoggle-style camera matching.
- Keep Blind Snapshot Duel the default mode and live 1v1 clearly optional.
- Clarify privacy, safety, and independence messaging.
- Collect public feedback through GitHub Issues and Discussions.

## Next

- Improve onboarding around camera permission and snapshot review.
- Add clearer public notes for leaderboard behavior and moderation expectations.
- Expand FAQ content for users comparing SnapMog with live-first alternatives.
- Improve support routing for account, payment, and safety questions.

## Later

- Evaluate additional duel formats after snapshot-first behavior is validated.
- Add public changelog entries for product-facing changes.
- Publish more guidance for safe and respectful camera duel behavior.

## Not In Scope

- Copying the production application source into this repository.
- Claiming affiliation with or endorsement by Omoggle.
- Handling private media, identity, billing, or account data through public issues.
